# PhD Journal Tracking

For day to day recording of PhD progress
- Use "entry x" where x can be:
  - Monday
  - last-monday
  - now
  - tomorrow
  - yesterday
  - yyyy-mm-dd

### Search all files in a directory with context (<mark>LINUX</mark>)

grep -r -C 3 "string" 

where 3 is the number of lines of context
